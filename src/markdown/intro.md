# 介绍

Cora UI 是一款基于 Vue 3 和 TypeScript 的 UI 组件库。提供了完整的类型提示，使用体验流畅便捷。

这款组件库完全基于最新的技术栈开发，所有组件均从零手写实现，不依赖任何第三方 UI 库。源码结构清晰，编码规范统一，每个组件的实现都简洁明了，易于阅读和理解。提供了完整的文档系统，每个组件都有详细的使用说明和可运行的示例代码，帮助开发者快速上手。

源代码已托管在 GitHub：https://github.com/hututu193/my-ui 你可以直接浏览每个组件的源码和示例，具体运行方法请参考仓库中的 README.md 文件。
